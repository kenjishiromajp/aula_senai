canvas
---------
canvas é uma nova tag HTML que surgiu com HTML5 que serve para desenhar gráficos via javascript.
existe uma série de métodos para desenhar gráficos

Mas afinal o que é canvas?
vocês já ouviram falar de HTML5? HTML5 é a nova versão do HTML, a mais atual e com ela vieram uma série de tags novas que não existiam. E canvas é uma delas.
Vocês sabem o que significa canvas? significa tela de pintura e é exatamente isso que ela é, mas ao envés de utilizar um pincel para desenhar, utilizamos o javascript. 
Mas para que ele é importante?, com ele podemos gerar gráficos de relatórios de acordo com algum dado, ou podemos criar algum jogo. Mostrar exemplos: Mostrar gráficos, mostrar sites com animações e etc.
